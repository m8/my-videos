## pre
- create `default.js` under `db` folder, template should be like:

```js
module.exports = {
   
    db: {
      user: "mysql_user_name",
      password: "mysql_user_password"
    },

};
```

## running
- you can run node with nodemon. to install `npm install -g nodemon`, after just type `nodemon` to run.
